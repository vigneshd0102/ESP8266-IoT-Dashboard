# 🚀 ESP8266 IoT Dashboard

## 📌 Overview
This project is a real-time IoT dashboard built using ESP8266. It provides device control, live monitoring, and data visualization through a web interface.

## 🔥 Features
- 🌐 Embedded Web Server
- 📡 REST API (`/status`, `/control`)
- 🔄 Real-time data updates (AJAX)
- 📊 Live sensor graph (Chart.js)
- ⏱ Uptime monitoring
- 🎨 Modern responsive UI
- 👤 Integrated developer profile section

## 🛠 Tech Stack
- ESP8266 (NodeMCU)
- Arduino IDE
- Embedded C++
- HTML, CSS, JavaScript
- Chart.js

## 📡 API Endpoints
- `/` → Dashboard UI  
- `/status` → JSON data  
- `/control?state=on/off` → Device control
